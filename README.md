# SRG Papers Bot

Monitors new scientific papers that use ART-XC data or Russian consortium eROSITA data and sends them to Telegram channels.

## Telegram channels

- [@srg_artxc](https://t.me/srg_artxc) — ART-XC papers
- [@srg_erosita](https://t.me/srg_erosita) — eROSITA papers (Russian consortium)


## GitHub Actions

Runs daily at 11:00 MSK via GitHub Actions.
